# DDAB

Public repository for hosting code associated with *Purposeful Bias: Algorithmic Bias Under Ideal Data Conditions*.  

1) knn_replication.R is used to replicate the results of Sections 4.B and 4.C.
2) svm_decision_function.R is used to replicate the results of Sections 4.A and 4.C.
3) SVM_Largen_Larged.R is used to replicate the results of Appendix A.
4) purposeful bias knn.Rmd can be used to recreate the visualizations in Section 4.B
5) Purposeful Bias.Rmd can be used to recreate the visualizations in Section 4.A.
6) CV_SVM.R is extends svm_decision_function.R to include the cross validation presented in Appendix B.
7) svm_protected_feature.r is used to replicate results of Appendix C.

*Note:  All code was run on Rstudio Version 1.4.1103 with R version 4.2.3 "Shortstop Beagle".  Different iterations of R/Studio may experience slight variations in random number generation even for a consistent set seed.*
